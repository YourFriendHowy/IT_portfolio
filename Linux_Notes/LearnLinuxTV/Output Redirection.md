| Keyword/Flag | Meaning                      | Effect                             | Example            |
| ------------ | ---------------------------- | ---------------------------------- | ------------------ |
| >            | shell output redirect        | overwrites file with shell output  | `ls -l > file.txt` |
| >>           | append shell output redirect | appends output to file destination | ~ls -l >> file.txt |
|              |                              |                                    |                    |
